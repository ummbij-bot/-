# 🏠 역할 2: 메인 대시보드 개발자

너는 **GoldenWalk (마실)** 프로젝트의 메인 화면 개발자야. 사용자가 앱을 켜자마자 보게 될 가장 중요한 대시보드를 구축해야 해.

## 🛠 작업 파일
- `app/page.js`: 메인 대시보드 화면

## 📌 구현 내용
1. **상단 인사**: "안녕하세요, 김영희님 👋" + 날씨 위젯
2. **걸음수 원형 그래프**: `CircleProgress` 컴포넌트를 사용하여 오늘의 걸음수(예: 2,847 / 4,000) 표시
3. **포인트 요약**: 현재 보유 포인트(예: 1,280P)를 강조한 카드
4. **주변 혜택 매장**: 가로 스크롤(`h-scroll`)을 이용해 `StoreCard` 목록 나열
5. **가족 응원 메시지**: 핑크색 톤의 아기자기한 가족 응원 섹션

## 📌 핵심 지침
- 시니어 사용자가 한눈에 오늘 몇 보 걸었는지 알 수 있게 디자인해.
- `app/globals.css`의 `page-content` 내부에서 레이아웃을 잡아.
- `BottomBar` 컴포넌트를 하단에 배치해.

### 예시 데이터
- 매장A: 종묘 교보약국, 300m, 50P
- 매장B: 을지 카페, 500m, 30P
- 응원: "딸 수진님이 ❤️ 보냈어요!"
