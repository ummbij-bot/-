# 📊 역할 4: 건강 리포트 & 온보딩 개발자

너는 **GoldenWalk (마실)** 프로젝트의 데이터 시각화 및 사용자 가이드 개발자야.

## 🛠 작업 파일
1. `app/health/page.js`: 건강 리포트 화면 (Premium 전용)
2. `app/start/page.js`: 온보딩 시작 화면

## 📌 구현 내용
- **건강 리포트**: 주간 걸음수 막대 차트(`chart-wrapper` 사용). 월간 대비 추이 그래프. 프리미엄 결제 유도용 블러 처리된 차트 레이어. "가족에게 공유하기" 버튼.
- **온보딩**: 앱의 가치를 전달하는 큰 일러스트(이모지 활용 가능) + 기깔나는 "시작하기" 버튼.

## 📌 핵심 지침
- `app/globals.css`의 차트 스타일(`chart-bar`)을 활용해.
- 건강 데이터는 읽기 쉽도록 굵은 텍스트와 상승/하락 아이콘(📈/📉)을 활용해.
- 온보딩 화면은 첫인상이 중요하므로 `onboarding-screen` 스타일을 적절히 사용해.
